# 个人导游网站

一个使用 UI/UX Pro Max 技能设计的精美个人导游网站。

**作者：zentrix566**

## 技术栈

- [Next.js](https://nextjs.org) 16
- Tailwind CSS
- Lucide 图标

## 功能特性

- **首页** - 高对比度首屏区域，三张带发光边框的服务卡片，专业的价格表
- **登录页** - 简洁的静态登录页面，支持社交登录选项
- 借鉴 Linear.app 的极简美学风格
- 深蓝色背景配白色文字与金色点缀

## 开始使用

```bash
npm run dev
# 或者
yarn dev
# 或者
pnpm dev
```

在浏览器中打开 [http://localhost:3000](http://localhost:3000) 查看结果。

## 页面

- `/` - 首页
- `/login` - 登录页
